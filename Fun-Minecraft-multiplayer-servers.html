Minecraft Servers

A Minecraft Server is something that allows
you to play on Minecraft with your friends. If you play PC Minecraft like I do, then you click the button called multiplayer. Then click the add server button and type the IP address in at the bottom.

  • Here are some of my favorite Minecraft Servers
  • Hive IP hivemc.net
  • cubecraft IP play.cubecraft.net
  • mineplex IP us.mineplex.com
  • high pixel IP play.hypixel.net
  • There are my favorite Servers! Have fun!!! :)

You can also use LAN. LAN is like a server, but you don't need a server address.
It uses wifi to connect you with your friends. Note you have to be on the same wifi
and you have to be on at the same time. If you are interested, here is how to do it.

  1. Get on the same wifi
  2. If you are on a PC, have one person get on
    singerplayer and the other one go on multiplayer and
    world will pop up :)

If you are a tablet player, you're in luck because you can do this too!

  1. Connect to the same wifi
  2. Have one person go in singleplayer and have the other player look in friends. You will see the other person's name pop up, and then click it, and you're in!!!
